# FinPilot AI

## AI-Powered Personal Finance Copilot

FinPilot AI is a modern full-stack personal finance management platform that helps users track income, expenses, investments, goals, family finances, net worth, and financial health using AI-powered insights.

The platform combines traditional finance management with AI-driven analysis to provide actionable recommendations for budgeting, saving, investing, and long-term wealth creation.

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/finpilot-ai.git
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

# Environment

Create a .env file inside backend:

```env
PORT=5000
JWT_SECRET=your_secret_key
```
